# شرح Plugin: Explanatory Output Style

## الفكرة الأساسية
Plugin يضيف شروحات تعليمية أثناء كتابة الكود.

---

## الوظائف

### 🧠 Insights
- 2-3 نقاط تعليمية
- مرتبطة بالكود

---

### 📦 Output Format
★ Insight
- explanation
- patterns
- trade-offs

---

### 🎯 Focus
- سبب الحل
- design patterns
- alternatives

---

### 🔄 Auto Mode
- يعمل تلقائيًا

---

## طريقة الاستخدام
- يعمل بدون أوامر

---

## المميزات
- تعلم أثناء العمل
- فهم أعمق
- تحسين التفكير
