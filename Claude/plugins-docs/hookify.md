# شرح Plugin: Hookify

## الفكرة الأساسية
Plugin لإنشاء قواعد تتحكم في سلوك AI باستخدام Markdown.

---

## الوظائف

### 🧠 Rules
- إنشاء قواعد بسهولة
- بدون كود

---

### ⚙️ Config
- YAML + Markdown
- regex support

---

### 🔍 Events
- bash
- file edits
- prompts
- session

---

### 🚫 Actions
- block
- warn

---

### 🔄 Live Updates
- تطبيق فوري

---

## طريقة الاستخدام
- `/hookify`
- `/hookify:list`
- `/hookify:configure`

---

## المميزات
- تحكم كامل
- منع الأخطاء
- تحسين الجودة

---

## الخلاصة
تحويل AI إلى نظام منضبط بقواعد مخصصة
