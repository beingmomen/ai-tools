# شرح Plugin: Firecrawl

## الفكرة الأساسية
Plugin لتحويل المواقع إلى بيانات نظيفة جاهزة للـ AI.

---

## الوظائف

### 🌐 Scrape
- استخراج صفحة واحدة
- Markdown / JSON

---

### 🕸️ Crawl
- تحليل موقع كامل

---

### 🔍 Search
- البحث + استخراج

---

### 🗺️ Map
- استخراج URLs

---

### 🤖 Agent
- جمع بيانات تلقائي

---

### 📦 Output
- Markdown
- JSON
- HTML

---

## طريقة الاستخدام
- `/firecrawl:setup`
- `/firecrawl:scrape`
- `/firecrawl:crawl`
- `/firecrawl:search`
- `/firecrawl:agent`

---

## المميزات
- scraping متقدم
- AI integration
- توفير وقت

---

## الخلاصة
تحويل الويب إلى بيانات قابلة للاستخدام مباشرة
