# شرح Plugin: Security Guidance

## الفكرة الأساسية
Plugin يحذر من مشاكل الأمان أثناء كتابة الكود.

---

## الوظائف

### 🛑 Auto Hook
- يعمل تلقائيًا قبل التعديل

---

### 🔍 فحص الأمان
- command injection
- XSS
- eval risks
- Python pickle
- GitHub Actions issues

---

### 💡 اقتراح حلول
- بدائل آمنة
- نصائح إصلاح

---

### 🔁 تقليل التكرار
- يظهر التحذير مرة واحدة

---

## طريقة الاستخدام
- يعمل تلقائيًا بدون أوامر

---

## المميزات
- حماية مستمرة
- تقليل الثغرات
- تحسين جودة الكود

---

## الخلاصة
تحويل كتابة الكود إلى تجربة أكثر أمانًا باستخدام تحذيرات ذكية
