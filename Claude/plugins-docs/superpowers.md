🔹 الفكرة الأساسية

إضافة Superpowers هي نظام بيعلّم الذكاء الاصطناعي (Claude) إزاي يشتغل زي مبرمج محترف، مش بس يكتب كود وخلاص.

يعني بدل ما يشتغل عشوائي:
👉 يشتغل بطريقة منظمة زي المهندسين الحقيقيين


🔹 بتعمل إيه؟

الإضافة دي بتخلي الـ AI يلتزم بأساليب تطوير قوية زي:

1. 🧠 التفكير قبل التنفيذ (Brainstorming)
يفهم المطلوب كويس الأول
يسأل ويحلل قبل ما يكتب كود
2. 🧪 TDD (Test Driven Development)

دي أهم فكرة 👇

يكتب الاختبارات الأول
الاختبار يفشل (red)
يكتب الكود (green)
يحسن الكود (refactor)

👉 يعني بيشتغل بطريقة منظمة جدًا مش عشوائية

3. 🐞 Debugging احترافي

مش مجرد يصلّح الغلط بسرعة
لا 👇

يحدد السبب الحقيقي للمشكلة
يجرب فرضيات
يحلل النمط
بعد كده يصلّح
4. 👨‍💻 Code Review تلقائي

فيه Agent بيراجع الكود:

هل الكود مظبوط؟
هل ماشي مع الخطة؟
هل فيه مشاكل تصميم؟
5. 🤖 Subagents

يعني يقدر يقسم الشغل:

جزء للتخطيط
جزء للتنفيذ
جزء للمراجعة

زي فريق كامل مش شخص واحد

🔹 طريقة الاستخدام

بتستخدم أوامر زي:

/brainstorming → للتفكير والتخطيط
/execute-plan → لتنفيذ الخطة خطوة خطوة
🔹 الفرق بينها وبين الإضافة الأولى
الأولى (Frontend Design) → تركيزها على الشكل والتصميم
دي (Superpowers) → تركيزها على طريقة الشغل والتنظيم

الإضافة دي هدفها:

تحويل الذكاء الاصطناعي من "يكتب كود بسرعة" → إلى "يشتغل زي مهندس سوفتوير محترف"
